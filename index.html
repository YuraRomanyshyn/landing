Responsive SLIDER

Hi, friends! this page was built to show that I can create responsive and adaptive web pages by using different technologies. Hope you will like it :)

P.S. Pictures that placed in RESPONSIVE SLIDER are examples of my designs.

Technologies i used here

HTML

Of course, I used html5 to create the "skeleton" of a page. I have good knowledge in markup language cause I had a lot of practice. Also, I collect my projects and sometimes use them or part of them as templates for my future projects.

COMPASS

Compass is a new technology for me, but it's pretty understandable. I like that it makes using Sass/ Scss easy to use. I'm learning this framework in more details cause it can be useful for me, in my opinion.

SCSS

SASS/SCSS is so cool! As always, I use clean CSS, but this preprocessor is awesome! Easy to make pages responsive and adaptive for different devices what is my target, actually.

JS

JavaScript is what I'm aiming at. So flexible language and so many possibilities. Every day I do steps to become a Pro from Junior user.

JQUERY

I use JQuery here for Responsive Tabs and Sliders. Cool library, but I think she will be useless in closest years. Who knows, we will see)

SUBLIME

I always use Sublime Text for coding, cause, in my opinion, it's the best editor ever made, even for my daily notes, not only for code)
